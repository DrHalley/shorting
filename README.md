# Shorting - URL Shortener 🚀

Shorting is a modern URL shortening service that allows you to shorten long URLs effortlessly. Built with Next.js, Prisma, and Tailwind CSS.

## 🌟 Features
- 🚀 **Fast and Easy to Use** - Shorten URLs with a single click.
- 📊 **Click Analytics** - Track the number of visits to shortened URLs.
- 🎨 **Modern and Sleek Design** - User-friendly interface with Tailwind CSS.
- 🔐 **Secure Links** - Secure storage mechanisms for database entries.

## 🚀 Usage
1️⃣ Enter a long URL on the homepage and click the **"Shorten"** button.  
2️⃣ The system will generate a shortened link for you.  
3️⃣ Copy and share the shortened link.

## 📌 Technologies Used
- **Next.js** - A modern React-based framework.
- **Prisma** - ORM and database management.
- **Tailwind CSS** - Fast and sleek design.
- **PostgreSQL** - Database management.
